# Trip Generation Workbooks

## Included Original Files

- `original_legacy/home_end_trip_generation_rates_lotus123.123`: source `HOMEGEN.123`.
- `original_legacy/paper_tables_lotus123.123`: source `TABLES.123`.
- `original_legacy/trip_generation_model_tables_lotus123.123`: source `TRIPGEN.123`.
- `original_legacy/trip_generation_tables_excel.xls`: source `TRIPTABS.xls`.

## Modernized Files

- `modernized/xlsx/*.xlsx`: LibreOffice conversions of the included legacy workbooks.
- `modernized/csv/*_values.csv`: value exports from the converted workbooks.
- `modernized/workbook_summary.json`: workbook dimensions, formula counts, checksums, and source review.

## Notes

The workbooks contain aggregate/derived trip-generation rates, coefficients, and paper tables. They do not include raw household survey microdata.
